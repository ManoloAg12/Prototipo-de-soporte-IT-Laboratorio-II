# Prototipo de Portal de Autoservicio de TI (Nivel 0)

## Descripción del Proyecto
Este repositorio contiene el código fuente del prototipo funcional para el Laboratorio II de la asignatura Gestión de Tecnologías de la Información de la Universidad Modular Abierta. El proyecto implementa una Base de Datos de Errores Conocidos (KEDB) diseñada bajo la arquitectura de servicios ITIL.

Su objetivo principal es lograr la deflexión de tickets hacia el Nivel 0, reduciendo el costo operativo de la mesa de ayuda al proporcionar a los usuarios herramientas de resolución autónoma para incidentes de alta frecuencia.

## Escenarios Técnicos Implementados
* **Conectividad:** Guía de resolución para fallas de autenticación en la red móvil institucional.
* **Hardware:** Procedimientos de sustitución menor e incidentes de impresión (atascos de papel y cambio de tóner).

## Características del Sistema
* **Autenticación de Usuarios:** Sistema de login simulado mediante validación de credenciales estáticas.
* **Motor de Búsqueda Semántica:** Filtro de catálogo basado en diccionarios de sinónimos para facilitar la ubicación de los Procedimientos Estándar (SOP).
* **Interfaz Fricción Cero:** Modales interactivos con instrucciones paso a paso redactadas en lenguaje estrictamente no técnico.

## Tecnologías Utilizadas
* HTML5
* Tailwind CSS (Framework)
* JavaScript Vanilla
